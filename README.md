# Smart Patrol JetBot

Embedded Systems project implementing a Finite State Machine (FSM) for autonomous patrol behavior.

## Architecture

- `fsm.py` – Core state machine logic
- `sensors/` – Vision and sensor inputs
- `actuators/` – Motor control logic
- `services/` – Alerting system
- `main.py` – Entry point

## Current Status

- FSM states implemented
- Project structured for hardware abstraction
- Preparing simulation environment integration
